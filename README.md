# scala-cli-millbuild-scalatest-hello-world

## Description
A POC for mill with scalatest
and scala.

## Tech stack
- scala
- scalatest
- mill

## Docker stack
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`
- test results
  - console
  - file
    - bin/out/HelloWorld/test/test/meta.json
    
## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
